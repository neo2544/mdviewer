# mdviewer — 사용 가이드 (TUI)

터미널에서 동작하는 마크다운 뷰어입니다. 좌측 사이드바에서 파일을 고르고, 우측 미리보기에서 내용을 확인합니다. 이 안내문은 선택된 파일이 없을 때 자동으로 표시됩니다.

---

## 실행 방법

```
mdviewer [경로]              # 해당 경로를 시작 디렉터리로 TUI 실행
mdviewer --web              # 웹 서버 모드 (브라우저에서 열림)
mdviewer --menubar          # macOS 메뉴바 모드 (웹 서버 + 메뉴바 아이콘)
mdviewer --port 8421        # 포트 지정 (--web / --menubar)
mdviewer --root <dir>       # 시작 디렉터리 명시
```

인수 없이 실행하면 현재 디렉터리에서 TUI가 시작됩니다.

## 기본 단축키

### 이동

| 키 | 동작 |
|----|------|
| `j` / `↓` | 아래 항목 |
| `k` / `↑` | 위 항목 |
| `Enter` | 디렉터리 진입 / 파일 미리보기 |
| `Tab` | 포커스 전환 (파일 목록 ↔ 즐겨찾기 ↔ 미리보기) |
| `Home` / `End` | 미리보기 처음/끝 |
| `PgUp` / `PgDn` | 미리보기 반쪽 페이지 이동 |

### 검색 / 경로 점프

| 키 | 동작 |
|----|------|
| `g` 또는 `:` | **Jump to path** — 경로를 입력하고 `Enter` 로 직접 이동 |
| `Esc` | Jump 입력 취소 |
| `Ctrl + U` | Jump 입력 비우기 |

> Jump는 절대 경로, `~` 시작 경로, 또는 파일 경로(해당 디렉터리로 이동 후 파일 선택)를 모두 지원합니다.

### 즐겨찾기

| 키 | 동작 |
|----|------|
| `f` | 현재 디렉터리를 즐겨찾기에 추가/제거 |
| `Tab` → `Favorites` 포커스 → `Enter` | 즐겨찾기 디렉터리로 점프 |

### 미리보기 새로고침 / 외부 변경 감지

| 키 | 동작 |
|----|------|
| `r` | 현재 미리보기를 강제로 다시 읽음 |
| `y` | 외부 변경이 감지됐을 때 — 다시 읽고 적용 |
| `n` | 외부 변경 알림을 일단 무시 (나중에 `r`/`y` 로 갱신 가능) |

mdviewer는 파일 시스템 변경을 백그라운드로 감시합니다. 미리보기 중인 파일이 외부에서 수정되면 상태 줄에 알림이 뜨고, 위 키로 어떻게 처리할지 선택할 수 있습니다.

### 종료

| 키 | 동작 |
|----|------|
| `q` | 종료 |
| `Ctrl + C` | 종료 |

## 화면 구성

```
┌──────────────────┬──────────────────────────────┐
│ Files            │ Preview                      │
│  ▶ file.md       │                              │
│    other.md      │   (마크다운 렌더링)          │
│                  │                              │
│ Favorites        │                              │
│    ~/notes       │                              │
└──────────────────┴──────────────────────────────┘
 status line: 현재 경로 / 외부 변경 알림 / 메시지
```

- **사이드바 상단**: 현재 디렉터리의 파일/하위폴더 목록 + 크기
- **사이드바 하단**: 즐겨찾기 디렉터리 목록
- **우측**: 선택된 파일의 마크다운 렌더링 결과
- **상태 줄**: 알림과 짧은 메시지

---

## 동작 팁

- 디렉터리 안에 너무 많은 파일이 있을 땐 `g` / `:` 로 직접 점프하는 게 빠릅니다.
- 외부 에디터로 자주 수정한다면 `y` 만 외워두면 충분합니다. 화면이 자동 동기화됩니다.
- 웹 UI(별칭, Recent files, Cmd+K 팔레트, mermaid 줌 등)가 필요하면 `mdviewer --web` 으로 실행하세요.

---

## 더 많은 기능이 필요할 때

웹 모드에는 다음이 추가로 있습니다:
- Recent files / Recent folders 영속 추적
- Favorites 별칭 (alias) 지정
- Cmd/Ctrl + K 단축키 팔레트
- Mermaid 다이어그램 줌 / pan / 텍스트 복사
- 직접 편집 + 저장

```
mdviewer --web --port 8421
```
