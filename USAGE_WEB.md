# 📘 Markdown Browser — Welcome

> 좌측에서 파일을 골라 읽고, 편집하고, 다이어그램을 보고, 자주 찾는 항목을 정리할 수 있는 **마크다운 뷰어**입니다.
> 이 안내문은 *파일이 선택되지 않았을 때* 자동으로 표시됩니다. 언제든 사이드바에서 ".." 로 올라가면 다시 만날 수 있어요. 👋

|------|-----------------|

## 🗂️ 1. 파일 탐색

- **사이드바**에서 디렉터리·파일을 클릭해 이동합니다. `..` 로 상위 폴더.
- **Search files** 입력창 — 현재 디렉터리에서 즉시 필터링 🔎
- **Jump to path** — 절대 경로, `~/...`, 파일 경로 모두 OK. `Enter` 로 점프 🚀

> 💡 `Cmd/Ctrl + L` 로 Jump 입력창에 바로 포커스 (브라우저 주소창과 비슷한 패턴).

## 📚 2. 사이드바 섹션 — Recent / Favorites

사이드바엔 항상 **최대 3개**만 노출돼서 화면을 깨끗하게 유지합니다. 더 보려면 헤더 우측의 **Show all (N)** 버튼!

### 📄 Recent files *(최대 50)*

직접 연 파일들이 가장 최근부터 쌓입니다. 팝업에선 다음 정보가 함께 나와요:

- 🕒 **Opened** — 마지막으로 열어본 상대시간 (호버하면 절대시각)
- 🏷️ **Status chip**
  - <kbd style="color:#e879c0">Updated</kbd> — 마지막 본 이후 외부에서 수정됨
  - <kbd style="color:#9ca3af">Up to date</kbd> — 변화 없음
  - <kbd style="color:#6b7280">—</kbd> — 이번 세션에 그 디렉터리에 안 들어가서 비교 불가

### 📁 Recent folders *(최대 20)*

직접 들어간 디렉터리들. 백그라운드 자동 새로고침은 기록되지 않습니다. 👌

### ⭐ Favorites + 별칭 (Alias)

- 현재 디렉터리를 Favorites에 추가/제거: 헤더의 **Toggle current** 버튼.
- 각 favorite에 **별칭(alias)** 을 붙일 수 있어요 ✍️
  - 행에 호버 → 우측 **✎** 버튼 클릭
  - 또는 행을 **더블클릭**
  - 빈 값 저장 → 별칭 제거
- 별칭이 있으면 **위에 별칭, 아래에 축약 경로** 두 줄로 표시됩니다.
- 별칭은 **서버에 저장**되어 다른 브라우저/기기에서도 보입니다! 🌍

## 🔔 3. 변경 알림 — 작은 컬러 점

파일명 오른쪽의 점이 변경 상태를 알려줍니다.

| 점 색 | 의미 |
|------|------|
| 🟦 청록 | **new** — 이전 스냅샷에 없던 새 파일 |
| 🟪 마젠타 | **updated** — modTime이 바뀐 파일 |
| 🟨 노랑 | **recent** — 처음 보는 디렉터리에서, 직전 세션 이후 수정된 파일 |
| ⭕ 빈 링 | 디렉터리 안의 **자식 중에 변경 사항이 있음** |

파일을 한 번 열어보면 그 점은 사라집니다. *Dismiss 상태가 영속화되어* 다시 안 켜져요. 😌

## ⚡ 4. 빠른 열기 — `Cmd/Ctrl + K`

`Cmd/Ctrl + K` 를 누르면 **Recent 팔레트**가 등장!

- 최근 연 파일/폴더가 통합 검색됩니다 (fuzzy 매칭) 🔎
- `↑ ↓` 이동, `Enter` 열기, `Esc` / 바깥 클릭 닫기
- 매칭 글자는 청록색으로 하이라이트 ✨

> 🛈 `Cmd + P` 는 macOS 인쇄 단축키와 충돌해서 `Cmd + K` 로 변경되었습니다.

## 🖼️ 5. 미리보기

- 📝 Markdown — 자동 렌더링
- 🌄 이미지 (PNG/JPG/GIF/SVG) — 그대로 표시, 클릭하면 라이트박스
- 🔄 우측 상단 **Refresh** — 디스크에서 다시 읽기

### 🎯 Mermaid 다이어그램 인터랙션

| 동작 | 단축키 |
|------|-------|
| 확대/축소 | 🖱️ **휠** (1배율 이상) · `Cmd/Ctrl + 휠` 강제 줌 |
| 화면 이동 | 🖐️ 드래그 (확대 상태에서) |
| 큰 화면으로 보기 | 🔍 **클릭** → 라이트박스 |
| 줌/위치 리셋 | 🔁 **더블클릭** |
| 다이어그램 전체 텍스트 복사 | 📋 호버 → **Copy text** 버튼 |
| 부분 텍스트 선택 | ⌨️ **Alt/Option** + 드래그 → 평소처럼 `Cmd/Ctrl + C` |

> 🛈 Alt 드래그 모드에선 다이어그램에 점선 외곽선이 뜨고 텍스트 커서로 바뀝니다.

## ✏️ 6. 편집 / 저장

- 우측 상단 **Edit** 버튼으로 편집 모드 진입 → **Preview** 로 복귀
- `Cmd/Ctrl + S` 로 저장 💾
- 외부에서 파일이 바뀌었는데 미저장 변경이 있으면 *충돌 감지 후 경고* ⚠️

## 🌗 7. 테마 (Auto / Light / Dark)

우측 상단 **테마 버튼**을 누를 때마다 순환:

```
Auto  →  Light  →  Dark  →  Auto
```

- **Auto** — 시스템 설정(다크/라이트 모드) 자동 추종 🌓
- **Light** — 항상 밝은 테마 ☀️
- **Dark** — 항상 어두운 테마 🌙
- 선택은 `localStorage` 에 저장됨 → 새로고침 후에도 유지 ✅

## 💾 9. 데이터는 어디에?

다음 정보는 브라우저 `localStorage` 에 저장되어 **같은 브라우저의 다음 세션에서도 유지**됩니다:

- 디렉터리별 파일 스냅샷 (변경 감지용)
- Dismiss한 변경 알림 (30일 TTL)
- 마지막 세션 종료 시각
- Recent files / Recent folders 큐
- 테마 선택

다음 정보는 **서버 측 파일**로 저장되어 **모든 브라우저/기기에서 공유**됩니다:

- ⭐ Favorites 디렉터리 목록 (`.mdviewer_favorites.json`)
- 🏷️ Favorites 별칭 매핑 (`.mdviewer_aliases.json`)

> 💡 브라우저 캐시를 비우면 localStorage 정보는 초기화됩니다. 서버 저장 정보(Favorites/Aliases)는 그대로 남아요.
