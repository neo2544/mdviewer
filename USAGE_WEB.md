# Markdown Browser — 사용 가이드 (Web)

좌측에서 파일을 골라 읽고, 편집하고, 다이어그램을 보고, 자주 찾는 항목을 정리할 수 있는 마크다운 뷰어입니다. 이 안내문은 파일이 선택되지 않았을 때 자동으로 표시됩니다.

---

## 1. 파일 탐색

- **사이드바**에서 디렉터리/파일을 클릭해 이동합니다. `..` 항목으로 상위 폴더로 갑니다.
- **Search files** 입력창에 글자를 치면 현재 디렉터리 안에서 즉시 필터링됩니다.
- **Jump to path** 입력창에 절대/홈(`~/...`) 경로나 파일 경로를 넣고 `Enter` 하면 그 위치로 직접 이동합니다.

> 단축키 `Cmd/Ctrl + L` 로 Jump to path 입력창에 바로 포커스할 수 있습니다 (브라우저 주소창 패턴).

---

## 2. 사이드바 영역들

사이드바엔 항상 최대 3개씩만 노출됩니다. 더 보려면 헤더 우측의 **Show all (N)** 버튼을 누르세요.

### Recent files (최대 50)
직접 연 파일들이 가장 최근부터 쌓입니다. Show all 팝업에서는 다음 메타가 함께 표시됩니다:
- **Opened**: 마지막으로 열어본 상대시간 (호버하면 절대시각)
- **Status chip**:
  - `Updated` — 마지막으로 열어본 이후 외부에서 수정됨
  - `Up to date` — 변화 없음
  - `—` — 그 디렉터리를 이번 세션에 안 열어봐서 비교 불가

### Recent folders (최대 20)
직접 들어간 디렉터리들이 누적됩니다. 자동 새로고침은 기록되지 않습니다.

### Favorites + 별칭 (Alias)
- 현재 디렉터리를 Favorites에 추가/제거: 헤더의 **Toggle current** 버튼.
- 각 favorite에 **별칭(alias)** 을 지정할 수 있습니다.
  - 행에 호버 → 우측의 **✎** 버튼 클릭
  - 또는 행을 **더블클릭**
  - 빈 값으로 저장하면 별칭이 제거됩니다.
- 별칭이 있으면 위에 별칭, 아래에 축약 경로가 두 줄로 표시됩니다.

---

## 3. 변경 알림 (점 표시)

파일명 오른쪽의 작은 컬러 점은 변경 상태를 의미합니다.

| 색 | 의미 |
|----|------|
| 청록 | **new** — 이전 스냅샷에는 없던 새 파일 |
| 마젠타 | **updated** — 이전에 봤던 파일의 modTime이 바뀜 |
| 노랑 | **recent** — 이 디렉터리를 처음 보는데, 직전 세션 이후 수정된 파일 |
| 빈 링(속 빈 점) | 디렉터리 안의 **자식 중에 변경 사항이 있음** |

파일을 한 번 열어보면 그 파일의 점은 사라집니다. 점을 끄기 위해 굳이 다시 안 열어도 되는 dismiss 상태가 영속화됩니다.

---

## 4. 검색 / 빠른 열기 — Cmd/Ctrl + K

`Cmd/Ctrl + K` 를 누르면 **Recent 팔레트**가 뜹니다.

- 최근 연 파일/폴더가 모두 통합 검색됩니다 (fuzzy 매칭).
- `↑ ↓` 로 이동, `Enter` 로 열기, `Esc` 또는 바깥 클릭으로 닫기.
- 검색어가 매칭된 글자는 청록색으로 하이라이트됩니다.

> `Cmd + P` 는 macOS 인쇄 단축키와 충돌해서 `Cmd + K` 로 변경되었습니다.

---

## 5. 미리보기

- Markdown 파일은 자동으로 렌더링됩니다.
- 이미지(PNG/JPG/GIF/SVG)는 그대로 표시되고, 클릭하면 라이트박스가 열립니다.
- 위쪽 **Refresh** 버튼은 디스크에서 다시 읽어옵니다.

### Mermaid 다이어그램

- **휠**: 1배율 이상에서는 확대/축소, `Cmd/Ctrl + 휠` 로 강제 줌
- **드래그**: 줌된 상태에서 화면 이동 (pan)
- **클릭**: 라이트박스로 크게 보기
- **더블클릭**: 줌/위치 리셋
- **호버 → 우측 상단 "Copy text"**: 다이어그램 내 모든 텍스트를 좌표 순서대로 합쳐 클립보드에 복사
- **Alt/Option + 드래그**: pan 대신 *텍스트 선택 모드*. 점선 외곽선이 뜨면서 SVG 내부 글자를 마우스로 끌어 선택 → 평소처럼 `Cmd/Ctrl + C`

---

## 6. 편집 / 저장

- 우측 상단 **Edit** 버튼으로 편집 모드. **Preview** 로 다시 미리보기.
- `Cmd/Ctrl + S` 로 저장.
- 외부에서 파일이 바뀌었는데 미저장 변경이 있으면 충돌을 감지하고 경고합니다.

---

## 7. 기타 단축키

| 키 | 동작 |
|----|------|
| `Cmd/Ctrl + K` | Recent 팔레트 열기/닫기 |
| `Cmd/Ctrl + L` | Jump to path 입력창 포커스 |
| `Cmd/Ctrl + S` | 저장 (편집 모드) |
| `Esc` | 팔레트/팝업/입력 취소 |
| `Alt/Option + drag` | Mermaid 텍스트 선택 모드 |
| `더블클릭` (favorite) | 별칭 편집 |

---

## 8. 데이터 / 영속화

다음 정보는 브라우저 `localStorage` 에 저장되어 다음 세션에서도 유지됩니다:
- 디렉터리별 파일 스냅샷 (변경 감지용)
- 사용자가 dismiss한 변경 알림 (30일 TTL)
- 마지막 세션 종료 시각 (recent 판정 기준)
- Recent files / Recent folders 큐
- Favorites 별칭

브라우저 캐시를 비우면 위 정보가 모두 초기화됩니다. Favorites 목록 자체는 서버 측에 저장되어 별개로 유지됩니다.
